# 67. Cut and Add

## 题目
字符串轮流执行：Harry 把末尾 `M` 个字符移到开头，Ron 把末尾 `N` 个字符移到开头。求最少多少次操作恢复原字符串。接口 `solution(string, m, n)`。

样例：`AbcDef,1,2` 输出 4；`abcabc,1,1` 输出 3。

## 解析
移动末尾字符等价于循环右移。只累计偏移量，并在每一步构造对应旋转与原串比较；轮次奇数使用 `m`，偶数使用 `n`。字符串可能有周期，因此不能只等待偏移回到零。

## 正确性与复杂度
累计偏移精确等于依次执行每次旋转的合成偏移，首次字符串相等的轮次就是最小操作数。至多检查 `2L` 个“偏移、轮到谁”状态。时间 `O(L^2)`，空间 `O(L)`。测试覆盖两组原题样例。
