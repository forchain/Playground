# 73. Roots of Quadratic Equation

## 题目
给定整数 `a,b,c`，求 `ax²+bx+c=0` 的两个整数根并以空格连接；题目保证根为整数。接口 `solution(a,b,c)`。

样例：`1,4,3` 输出 `-1 -3`；`1,2,1` 输出 `-1 -1`。

## 解析
按求根公式计算判别式 `D=b²-4ac`，使用整数平方根避免浮点误差，再计算 `(-b+sqrt(D))/(2a)` 和另一根。实现也验证题目承诺。

## 正确性与复杂度
求根公式对所有 `a != 0` 的二次方程成立，且整数运算精确得到题目保证的根。时间、空间均为 `O(1)`。根顺序与原题样例一致，测试覆盖两例。
