# Max Sum Circular Subarray

## 题目

给定非空环形整数数组，返回一个非空连续子数组的最大和。子数组可从数组末尾环绕到开头。`n <= 100`。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

答案分为不跨边界和跨边界两类。不跨边界由 Kadane 最大子数组得到；跨边界等于数组总和减去中间被排除的最小子数组。全负数组需单独返回普通最大子数组，防止错误选择空数组。

## 正确性说明

任何环形连续段若不跨边界已被第一类覆盖；若跨边界，其补集必是一个普通连续段，最大化所选和等价于最小化补集和。两类取最大穷尽全部合法答案。全负分支保证结果非空。

## 复杂度分析

时间 `O(n)`，空间 `O(1)`。

## 样例与测试

原题全部可执行样例已写入 `test_solution.py`：`[1,-2,3,-2] -> 3`；`[5,-3,5] -> 10`；`[-3,-2,-1] -> -1`。测试直接调用函数接口，避免输入展示换行差异影响算法验证。
