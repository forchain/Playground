# 84. Detect Cycle in a Linked List

## 题目
判断单链表是否有环，有环返回 `True`，否则 `False`。测试数据用末节点连接到的索引描述环，该索引不传入 `detectLoop(head)`。

样例：`[2,5,-4,3]` 末节点连索引 1、有环；`[1,2]` 连索引 0、有环；第三例给出节点值 5、索引 -1，无环。

## 解析
Floyd 快慢指针：慢指针每次一步、快指针两步。有环时二者最终相遇；无环时快指针抵达空指针。

## 正确性与复杂度
无环不可能重复节点，快指针必结束；有环后两指针在环内的相对距离每轮缩短一个模环长单位，必相遇。时间 `O(n)`、空间 `O(1)`。原文第三例声明 `N=2` 但只列一个值，测试按实际给出的单节点数据处理并明确注释。
