# 80. Countries and Capitals

## 题目
平台提供同步 `getCountry`、异步 `getCapitalFromCountry`、同步 `getPopulationFromCapital`。正常返回 `国家, 首都, 人口`；索引无国家返回 `No country found at ind = {index}`；国家无首都返回对应提示。

样例索引 0、1、2、10000 的结果分别为印度记录、美国记录、无首都提示、无国家提示。Python 接口通过参数注入三个平台函数。

## 解析
先同步查国家并判断空值，再等待首都；首都查询拒绝时返回规定提示，成功后查询人口并格式化。

## 正确性与复杂度
控制流与题目三种状态一一对应，正常路径中的三个字段均来自指定平台函数。忽略外部调用耗时，算法时间和空间 `O(1)`。异步测试覆盖四个原题样例。
