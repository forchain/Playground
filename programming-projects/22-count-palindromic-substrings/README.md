# Count Palindromic Substrings

## 题目

给定仅含小写字母、长度 1 到 1000 的字符串 `s`，返回其中不同回文子串的数量。子串必须连续；回文串正读与反读相同。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

以每个字符（奇数长度）及相邻字符间隙（偶数长度）为中心向两侧扩展。扩展期间遇到的每个回文片段加入集合，集合自动消除不同位置产生的相同文本。

## 正确性说明

每个回文子串都有唯一的中心，扩展到它的左右边界时必会加入集合；算法只在两端字符相等时加入，故集合中也不会出现非回文串。最终集合恰为全部不同回文子串。

## 复杂度分析

时间 `O(n^3)`（最多 `O(n^2)` 次扩展，Python 切片和哈希最长 `O(n)`），空间 `O(n^2)`。
