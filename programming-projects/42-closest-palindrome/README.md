# Closest Palindrome

## 题目

输入无前导零的正整数字符串（长度 1 到 18），返回数值距离最近且不等于自身的回文整数（字符串形式）；距离相同取较小者。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

回文由左半部分唯一决定。取原数左半前缀的 `prefix-1`、`prefix`、`prefix+1` 并镜像，同时加入位数变化边界 `99...9` 和 `100...001`。删除原数后按 `(绝对差, 数值)` 取最小。

## 正确性说明

固定长度且最接近原数的回文，其决定前缀不可能离原前缀超过 1，否则向原前缀靠近即可得到更近候选。跨位数最近候选只可能是全 9 或 1 后跟零再跟 1。候选集因而完备，排序规则同时实现最近与平局取小。

## 复杂度分析

时间 `O(L)`，空间 `O(1)`（候选数恒定，L 为数字长度）。
