# Merge Overlapping Intervals

## 题目

给定 `n` 个闭区间 `[start,end]`，合并所有重叠区间并返回按起点排列的二维数组。若前一区间终点大于等于后一区间起点，两者重叠。`1 <= n <= 100`。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

先按起点排序。扫描区间：若当前起点不超过结果末区间的终点，就扩大末区间终点；否则开始一个新区间。实现复制首区间，避免修改调用者数据。

## 正确性说明

排序后，与当前合并段有可能相交的区间必连续出现。若起点不超过当前终点，取两终点最大值恰为并集；否则后续区间起点更大，当前段再也不可能与后面相交，可安全定稿。

## 复杂度分析

时间 `O(n log n)`，空间 `O(n)`（返回结果及排序）。

## 样例与测试

原题全部可执行样例已写入 `test_solution.py`：`[[1,3],[2,6],[8,10],[15,18]] -> [[1,6],[8,10],[15,18]]`；`[[1,4],[4,5],[6,8]] -> [[1,5],[6,8]]`。测试直接调用函数接口，避免输入展示换行差异影响算法验证。
