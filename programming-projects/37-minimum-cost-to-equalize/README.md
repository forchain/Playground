# Minimum Cost to Equalize

## 题目

给定最多 100000 个正整数，每次把任意元素增减 1，成本为 1。求把所有元素变为同一整数的最小总成本。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

总成本是目标值与所有数组元素绝对差之和。排序后选取中位数作为目标，再求绝对差总和；偶数长度时两个中位数之间任意整数都最优，选择上中位数即可。

## 正确性说明

当目标从左向右跨过一个数据点时，左侧项的成本斜率增加、右侧项的成本斜率减少。到中位数处左右数量达到平衡，成本从下降转为上升，因此中位数取得全局最小值。

## 复杂度分析

时间 `O(n log n)`，空间 `O(n)`（排序副本）。
