# 83. Delete Node From Singly Linked List

## 题目
删除单链表第 `k` 个节点（1 基），返回修改后的头，保证 `1<=k<=N<=10000`。接口 `deleteNode(head,k)`。

样例：`2 4 5 7,k=3` 输出 `2 4 7`；`1 3 4,k=3` 输出 `1 3`。

## 解析
删除头节点时直接返回 `head.next`；否则前进到第 `k-1` 个节点，令其 `next` 跳过目标节点。

## 正确性与复杂度
除前驱指针外所有链接保持不变，而前驱直接连接目标后继，故恰好移除第 `k` 节点并保持顺序。时间 `O(k)`、空间 `O(1)`。测试覆盖原题两例。
