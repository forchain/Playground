# 76. Smallest Subarray

## 题目
求最短连续子数组长度，使它至少包含整个数组的一个最小值和一个最大值。`1 <= n <= 2000`，接口 `minSubarrayLength(n,arr)`。

样例：`[1,3]` 输出 2；`[2]` 输出 1；`[1,5,10,7,1,9,4]` 输出 3。

## 解析
先求全局最小、最大值。扫描时记录最近一次二者出现的位置；每当两者都存在，它们之间的区间就是以当前相关位置为端点的最短候选。

## 正确性与复杂度
任意最优区间的右端相关极值出现时，另一种极值最近的位置会产生不长于该区间的候选，因此扫描不会漏掉最优解。时间 `O(n)`，空间 `O(1)`。测试覆盖三例。
