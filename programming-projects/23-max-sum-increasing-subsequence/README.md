# Max Sum Increasing Subsequence

## 题目

给定长度 1 到 100、元素范围 -10^4 到 10^4 的整数数组，返回严格递增子序列的最大元素和。子序列保持原相对顺序但不要求连续。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

令 `dp[i]` 为以第 `i` 个元素结尾的严格递增子序列最大和，初值为该元素本身。枚举此前更小元素 `j`，用 `dp[j] + array[i]` 更新。

## 正确性说明

任一以 `i` 结尾的可行序列，要么只有 `array[i]`，要么倒数第二项位于某个满足 `array[j] < array[i]` 的 `j`。转移穷尽这些情况并取最大，归纳可知每个状态最优，所有结尾的最大值就是答案。

## 复杂度分析

时间 `O(n^2)`，空间 `O(n)`。

## 样例与测试

原题全部可执行样例已写入 `test_solution.py`：`[1,101,2,3,100,4] -> 106`；`[10,5,4,3,2] -> 10`；`[3,4,5,10,1,2,3] -> 22`。测试直接调用函数接口，避免输入展示换行差异影响算法验证。
