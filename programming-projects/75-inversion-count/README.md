# 75. Inversion Count

## 题目
数组中满足 `i<j` 且 `a[i]>a[j]` 的元素对称为逆序对，求逆序对数量。`2 <= n <= 10000`，接口 `solution(n,arr)`。

样例：`[2,4,1,3,5]` 输出 3；`[1,2,5]` 输出 0。

## 解析
归并排序合并两个有序半区时，若右侧当前值小于左侧当前值，它也小于左区尚未取出的全部元素，一次增加这些元素的数量。

## 正确性与复杂度
递归统计左右半区内部逆序对，合并阶段恰好统计所有跨半区逆序对，三类互斥且覆盖全部逆序对。时间 `O(n log n)`，空间 `O(n)`。测试覆盖两例。
