# 70. Perfect Numbers

## 题目
输出 1 到 `n` 中所有完全数，即除自身外因子之和等于自身的数，以空格连接；没有则返回空串。`2 <= n <= 1,000,000`，接口 `solution(n)`。

样例：`n=7` 输出 `6`；`n=200` 输出 `6 28`。

## 解析
建立真因子和数组。对每个可能因子 `d`，把 `d` 加到 `2d,3d,...`，最后筛选因子和等于自身的整数。

## 正确性与复杂度
每个数的每个真因子恰好在该因子的倍数循环中贡献一次，故数组值正是真因子总和。调和级数给出时间 `O(n log n)`，空间 `O(n)`。测试使用原题两例。
