# Minimum Swaps To Sort Array

## 题目

给定 `N` 个互异整数，允许交换任意两个位置，求升序排序的最少交换次数。`1 <= N <= 100`。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

排序后建立每个值的目标下标，当前位置到目标位置形成一个置换。遍历置换的各个环，长度为 `m` 的环累加 `m-1`。

## 正确性说明

一个长度 `m` 的非平凡环至少要改变 `m-1` 个错位关系，少于 `m-1` 次交换无法全部归位；固定一个位置并依次换回其余元素可恰好用 `m-1` 次。各环独立，求和即全局最少。

## 复杂度分析

时间 `O(n log n)`，空间 `O(n)`。

## 样例与测试
