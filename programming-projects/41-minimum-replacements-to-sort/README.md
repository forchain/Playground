# Minimum Replacements to Sort

## 题目

给定正整数数组。一次操作可把一个元素替换为两个和等于原值的元素，求使整个数组非递减所需的最少操作。`n <= 100`，元素不超过 10^9。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

从右向左维护当前元素各拆分份允许的最大值 `upper_bound`。若值过大，至少拆成 `ceil(value/upper_bound)` 份，操作数为份数减一；尽量均分后，最左份最大值为 `floor(value/parts)`，成为下一轮上界。

## 正确性说明

为了不超过右侧上界，份数少于该下取整上取整值必有一份超限，因此所选份数最少。固定最少份数后，均分能最大化最小份（即给左侧的上界），不会使后续更差。由右向左归纳，贪心操作数全局最少。

## 复杂度分析

时间 `O(n)`，空间 `O(1)`。
