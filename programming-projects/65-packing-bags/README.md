# 65. Packing Bags

## 题目
每个购物袋容量为 10kg，物品重量均为 1 到 10kg 且可任意排序。给定物品数 `n`、重量数组和袋子数，判断是否能装下全部物品。接口为 `solution(n, items, bags)`，返回布尔值。

样例：`[2,1,2,5,4,3,6,1,1,9,3,2]` 与 4 个袋子返回 `true`；`[2,7,1,3,3,4,7,4,1,8,2]` 与 4 个袋子返回 `false`。

## 解析
总重量不超过总容量只是必要条件，不能保证能分袋。将物品从重到轻放置并回溯，记录每袋剩余容量。相同剩余容量的袋子等价，因此每层只尝试一次同容量状态，可显著剪枝。

## 正确性与复杂度
回溯枚举每件物品进入每个可容纳它的袋子，覆盖所有合法分配；找到完整放置即返回真，穷尽仍失败即不存在方案。最坏时间为 `O(bags^n)`，空间为 `O(n + bags)`。测试覆盖原题两个样例。
