# Permutation Sequence

## 题目

给定 `1 <= n <= 9` 和 `1 <= k <= n!`，返回集合 `[1,2,...,n]` 的第 k 个字典序排列，结果为字符串。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

将 k 减一转为零基排名。首位每个候选对应 `(n-1)!` 个连续排列，用整除确定候选下标、取余得到块内排名；删除候选后对余下位置重复。

## 正确性说明

字典序中，以相同首元素开头的排列形成大小相同且连续的块。整除精确选出第 k 项所属块，余数给出其在块内排名；逐位应用相同论证，最终唯一确定目标排列。

## 复杂度分析

时间 `O(n^2)`（列表中间删除），空间 `O(n)`。
