# Find Longest Repeating Substring

## 题目

给定长度不超过 1000 的小写字符串，求至少出现两次且两次出现不能重叠的最长子串长度；不存在返回 0。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

`dp[i][j]` 记录分别以 `i-1`、`j-1` 结尾的相同后缀长度。字符相等时从左上状态加一，同时用两个结尾的距离 `j-i` 截断，从而保证两段不重叠。

## 正确性说明

相等后缀只能由前一对相等后缀延长；截断后长度不超过两结尾间距，故不会重叠。反之任意两段不重叠的相同子串，其末字符状态会逐字符延长到该长度，因此答案不会遗漏。

## 复杂度分析

时间 `O(n^2)`，空间 `O(n^2)`。
