# Longest Palindromic Subsequence

## 题目

给定长度不超过 1000 的小写字符串，求最长回文子序列长度。子序列可删除字符但不能改变余下字符相对顺序。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

区间 DP：两端相等时答案为内部区间加 2；否则舍弃左端或右端并取较大者。实现按左端从右向左滚动，用 `diagonal` 保存更新前的左上状态。

## 正确性说明

若区间两端相同，存在包含两端的最优回文结构；若不同，任意回文子序列不可能同时使用两个不同端点，必包含于去掉其中一端的某个子区间。转移覆盖最优情况，按区间长度归纳成立。

## 复杂度分析

时间 `O(n^2)`，空间 `O(n)`。
