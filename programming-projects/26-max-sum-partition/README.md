# Max Sum Partition

## 题目

把整数数组分成任意多个非空连续段，目标最大化各段最大值之和。`1 <= N <= 100`，元素范围 -10^4 到 10^4。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

每个正数单独成段会完整贡献自身，合并不会得到更高贡献；非正数独立成段会降低总和，因此只需让它们并入相邻段。若没有正数，则整段只贡献数组最大值。

## 正确性说明

有正数时，把正数分别独立并将所有非正数附着到某些段，可取得所有正数之和。任意分段中，每段最大值至多等于该段正数之和（无正数段贡献非正），所以不可能超过该值。全非正时，减少段数不会降低单段最大值之和的上界，整段最大值最优。

## 复杂度分析

时间 `O(n)`，空间 `O(1)`。
