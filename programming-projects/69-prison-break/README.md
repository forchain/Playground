# 69. Prison Break

## 题目
0 表示牢房已解锁，1 表示锁定。从最左端向右走，只能释放当前实际为 0 的牢房；每释放一次，所有牢房状态全部翻转，且不能回头。若首格锁定则无法开始。求释放人数，接口 `countReleasedPrisoner(cells)`。

样例：`[1,1,1]` 输出 0；`[0,0,1,1,1,0,1]` 输出 4。

## 解析
无需真的反转整个数组，只维护全局翻转奇偶 `flipped`。当前实际状态是 `cell XOR flipped`；为 0 时释放并切换奇偶。

## 正确性与复杂度
奇偶变量准确表示此前所有全局翻转的合成效果，因此每个位置的判断与逐次翻转数组完全一致。单次扫描，时间 `O(n)`、空间 `O(1)`。测试覆盖全部原题样例。
