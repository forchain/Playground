# 77. Pick the Points

## 题目
给定若干闭整数区间，选择最少整数点，使每个区间至少包含一个所选点。接口 `solution(numOfsets,arr)` 返回点数，题目保证至少两个区间。

样例：`[1,3],[2,5],[3,6]` 输出 1；`[4,7],[1,3],[2,5],[5,6]` 输出 2。

## 解析
按右端点升序处理。遇到未被已选点覆盖的区间，就选它的右端点；这个位置在覆盖当前区间的同时，对之后区间最有利。

## 正确性与复杂度
任一方案必须为最早结束的未覆盖区间选一个点，将该点替换为此区间右端点不会失去对后续区间的覆盖。反复交换得到贪心方案且点数不增，因此最优。时间 `O(n log n)`，空间 `O(n)`。
