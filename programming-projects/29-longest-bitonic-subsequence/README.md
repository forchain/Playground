# Longest Bitonic Subsequence

## 题目

求整数数组中先严格递增再严格递减的最长子序列长度，元素不要求连续但保持相对顺序。`1 <= n <= 100`。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

正向动态规划得到以每个位置为峰尾的最长递增长度 `inc`，反向动态规划得到从每个位置开始的最长递减长度 `dec`。枚举峰值，合并为 `inc[i]+dec[i]-1`。

## 正确性说明

任一双调子序列存在峰值；峰前不长于该位置的最优递增状态，峰后不长于最优递减状态。反过来，这两个状态共享峰值并拼接后必为合法双调子序列，故枚举峰值所得最大值恰为最优。

## 复杂度分析

时间 `O(n^2)`，空间 `O(n)`。

## 样例与测试

原题全部可执行样例已写入 `test_solution.py`：`[1,2,5,3,2,1] -> 6`；`[1,11,2,10,4,5,2,1] -> 6`；`[12,11,40,5,3] -> 4`。测试直接调用函数接口，避免输入展示换行差异影响算法验证。
