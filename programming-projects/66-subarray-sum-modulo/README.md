# 66. Subarray Sum Modulo

## 题目
给定 `n` 个正整数、模数 `m`，求任意连续子数组元素和对 `m` 取模后的最大值。`2 <= n <= 10^5`，`m <= 10^14`。接口 `solution(n, m, arr)`。

样例：`[3,3,9,9,5], m=7` 输出 `6`；`[1,2,3], m=6` 输出 `5`。

## 解析
设前缀模为 `p[j]`，子数组模为 `(p[j]-p[i]) mod m`。对当前前缀，答案要么是它自身，要么来自此前严格大于它的最小前缀，使绕模后的值最大。先压缩所有前缀值，再用树状数组维护已出现值并查询严格后继。

## 正确性与复杂度
所有子数组都对应一对前缀；对固定右端点，严格大于当前值的最小旧前缀产生最大绕模差，因此逐点取最大覆盖全局最优。树状数组查询与更新均为对数时间，总时间 `O(n log n)`、空间 `O(n)`；样例测试均来自原题。
