# Maximal Rectangle Area

## 题目

给定 `n x m` 的 0/1 矩阵，返回只包含 1 的最大轴对齐矩形面积。`1 <= n,m <= 100`。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

逐行把矩阵转化成柱状图：`heights[c]` 是当前行向上连续 1 的高度。对每行使用单调递增栈，元素出栈时以该高度为最矮柱计算能延伸的最大宽度。

## 正确性说明

任意全 1 矩形都有一条底边所在行，并对应该行柱状图中的连续柱区间。单调栈在区间右边界首次遇到更矮柱时，会以区间最矮柱精确计算其最大宽度，因此不会遗漏最优矩形；计算出的每个面积也都由连续足够高的柱构成，必合法。

## 复杂度分析

时间 `O(nm)`，空间 `O(m)`。
