# Maximal Square Area

## 题目

给定 `n x m` 的 0/1 矩阵，返回只包含 1 的最大正方形面积。`1 <= n,m <= 100`。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

令状态为以当前格作右下角的最大全 1 正方形边长。当前格为 1 时，它等于左、上、左上三个状态最小值加一；为 0 时归零。实现用一维数组滚动。

## 正确性说明

边长大于 1 的正方形成立，当且仅当其左、上、左上相邻正方形都至少有边长 `k-1` 且当前格为 1；最短的相邻状态决定可扩展边长。故转移既不会构造非法方形，也能构造所有合法方形。

## 复杂度分析

时间 `O(nm)`，空间 `O(m)`。
