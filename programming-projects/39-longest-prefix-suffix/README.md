# Longest Prefix Suffix

## 题目

给定长度不超过 1000 的小写字符串，返回同时是前缀和后缀且两次出现不重叠的最长长度。不存在返回 0。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

先用 KMP 前缀函数得到整个字符串的最长 border（相等真前缀和后缀）。若长度超过字符串一半，沿前缀函数失配链退到次长 border，直至满足 `2*length <= n`。

## 正确性说明

KMP 前缀函数给出最长 border，且该 border 的所有更短候选恰沿失配链出现。非重叠等价于长度不超过 `n/2`，因此沿候选从长到短找到的第一个满足者就是答案。

## 复杂度分析

时间 `O(n)`，空间 `O(n)`。
