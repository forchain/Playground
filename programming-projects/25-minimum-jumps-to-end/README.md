# Minimum Jumps to End

## 题目

数组 `arr[i]` 表示从下标 `i` 最远可向前跳多少步。求从首项到最后一项的最少跳数；不可达返回 -1。`1 <= n <= 100`，`0 <= arr[i] <= 100`。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

按跳数做隐式分层扫描：`current_end` 是当前跳数能到的最右位置，`farthest` 是扫描本层时下一跳能到的最右位置。扫描到层尾就增加一次跳跃并进入下一层；无法扩大边界即不可达。

## 正确性说明

当前层覆盖恰好是使用既定跳数能到达的所有位置。对其全部位置取最远下一端点，就得到再跳一次可达范围；因此第一次覆盖末尾时使用的跳数最少。边界不再扩张时不存在后继路径。

## 复杂度分析

时间 `O(n)`，空间 `O(1)`。

## 样例与测试

原题全部可执行样例已写入 `test_solution.py`：`[2,3,1,1,4] -> 2`；`[1,0,0,0,0] -> -1`；`[0] -> 0`。示例 1 的解释误写“index 5”，数组末项实际为 index 4，不影响期望值。。测试直接调用函数接口，避免输入展示换行差异影响算法验证。
