# Longest substring without repeating characters

## 题目

给定至少含一个字符的字符串 `s`，返回不含重复字符的最长连续子串长度。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

维护滑动窗口左边界和每个字符最后出现位置。扫描右端字符时，若其旧位置仍在窗口内，就把左边界移到旧位置后一位；随后更新位置和最大窗口长度。

## 正确性说明

每次移动后，窗口中当前字符只有最新一次，其他字符此前已保持唯一，因此窗口始终合法。左边界只移动到消除冲突所需的最小位置，所以对每个右端点，当前窗口是能取得的最长合法窗口；全局最大即答案。

## 复杂度分析

时间 `O(n)`，空间 `O(字符集大小)`。

## 样例与测试

原题全部可执行样例已写入 `test_solution.py`：`abcabcbb -> 3`；`abcda -> 4`；`aaacbb -> 3`。测试直接调用函数接口，避免输入展示换行差异影响算法验证。
