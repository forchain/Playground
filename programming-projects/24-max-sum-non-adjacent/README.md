# Max Sum Non-Adjacent

## 题目

给定整数数组，选择若干互不相邻的元素，使总和最大；允许不选任何元素，因此空数组或全负数组可返回 0。约束为 `n <= 100`、元素绝对值不超过 10^4。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

逐项维护 `skip`（上一项未选）和 `take`（上一项已选）两种最优值。处理新元素时，选择它只能来自旧 `skip`；不选它可来自旧状态的较大者。

## 正确性说明

任何合法选择对当前元素只有选或不选两类，且选当前元素时上一元素必须不选。转移覆盖且不混淆这两类，归纳后两状态分别最优，最终取最大即全局最优。

## 复杂度分析

时间 `O(n)`，空间 `O(1)`。

## 样例与测试

原题全部可执行样例已写入 `test_solution.py`：`[3,2,7,10,12] -> 22`；`[-1,2,9,-4] -> 9`；`[5,5,10,100,10,5] -> 110`。测试直接调用函数接口，避免输入展示换行差异影响算法验证。
