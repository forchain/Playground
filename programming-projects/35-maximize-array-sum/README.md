# Maximize Array Sum

## 题目

给定整数数组和整数 `K`，至多执行 K 次操作，每次可将一个元素取相反数，求最大数组和。`N <= 100`，`K <= 10^4`。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

按值排序后，从最负的元素开始翻转，直到没有负数或次数用完。由于题意是“至多”K 次，剩余操作无需强行使用，也无需对正数或零进行无收益翻转。

## 正确性说明

翻转负数 `x` 会让总和增加 `-2x > 0`，且越小的负数增益越大；翻转非负数不会增加总和。因此选择至多 K 个最小负数是最优选择，交换论证可知任何跳过更小负数的方案都不优。

## 复杂度分析

时间 `O(n log n)`，空间 `O(n)`。
