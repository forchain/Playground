# 68. Partition Array

## 题目
从长度 `n` 的数组中选一个元素，使其值等于其余所有元素的乘积，返回最小的 0 基索引；无解返回 -1。接口 `solution(n, arr)`，`2 <= n <= 100`。

样例：`[2,8,4,1]` 输出 1；`[2,2,8]` 输出 -1。

## 解析
按索引从小到大枚举候选，直接计算除该位置外的乘积。这样自然处理 0、负数和重复元素，也天然保证返回最小索引。

## 正确性与复杂度
算法逐一检验所有且仅有的候选位置，首个满足定义者必为最小索引；全部失败即无合法划分。时间 `O(n^2)`，额外空间 `O(1)`。测试使用原题两例。
