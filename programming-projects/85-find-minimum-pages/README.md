# 85. Find Minimum Pages

## 题目
按原顺序把 `n` 本书分给 `m` 名学生，每人至少一本且拿连续书段，使单个学生页数最大值尽可能小；无法分配返回 -1。接口 `solution(n,m,arr)`。

样例：`[12,34,67,90],m=2` 输出 113；`[15,17,20],m=2` 输出 32。

## 解析
答案位于最大单本页数与总页数之间。二分最大允许页数；可行性检查从左到右贪心，当前学生放不下下一本时才换人，从而用最少学生完成该上限。

## 正确性与复杂度
贪心检查若需超过 `m` 人，则任何切法都不可行；若至多 `m` 人，可进一步切分为恰好 `m` 个非空连续段。可行性随上限单调，二分得到最小可行值。时间 `O(n log(sum(arr)))`，空间 `O(1)`。测试覆盖两例。
