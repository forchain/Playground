# Longest Common Subsequence

## 题目

给定两个长度不超过 1000 的小写字符串，返回最长公共子序列长度。公共子序列在两个字符串中保持相对顺序，不要求连续。

函数接口：`solution(...)`，直接返回题目要求的结果，不读取标准输入。

## 详细解析

经典二维状态压缩为一维：字符相等时由上一行左上角加一；不等时取上一行同列与当前行左列的较大值。令较短字符串作为列以节省空间。

## 正确性说明

两末字符相同，可把它接到两前缀的公共子序列后；不同则最优解至少舍弃其中一个末字符。转移穷尽两类情况，按前缀长度归纳得到每个状态最优。

## 复杂度分析

时间 `O(|s1||s2|)`，空间 `O(min(|s1|,|s2|))`。
