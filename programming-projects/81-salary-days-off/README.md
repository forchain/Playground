# 81. Salary Days Off

## 题目
员工字典包含 `id、salary、manager、num_days_off`。返回二元组：全体最高工资，以及所有经理休假天数之和。接口 `solution(dataset)`。

题目示例三名员工工资为 3000、5000、6000，后两人为经理且休假 10、14 天，输出 `(6000,24)`。

## 解析
一次聚合求所有工资最大值；另一次只筛选 `manager` 为真的记录并累加休假天数。

## 正确性与复杂度
最大值遍历全体员工，求和遍历且仅包含经理，因此两个分量分别满足定义。时间 `O(n)`，生成器额外空间 `O(1)`。测试使用原题完整示例数据。
