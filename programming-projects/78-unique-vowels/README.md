# 78. Unique Vowels

## 题目
给定小写字符串数组和整数 `n`，统计恰好包含 `n` 种不同元音的字符串数。接口 `solution(array_str,n)`。

原题表格样例：`eiqwa aaabc aheo iou,3` 输出 3；`aa bhg aeo lia,2` 输出 1；`anhj berq lopa,1` 输出 2。

## 解析
对每个字符串取字符集合，与元音集合 `aeiou` 求交；交集大小就是不同元音种数，等于 `n` 时计数。

## 正确性与复杂度
集合交集包含且只包含字符串出现过的不同元音，所以判断与题意等价。总时间和空间上界均为所有字符串总长度 `O(L)`。测试覆盖表格全部三例。
